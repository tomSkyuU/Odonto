🦷 Site Clínica Odontológica 📄 Sobre o projeto

Este projeto é um site institucional para uma clínica odontológica, com o objetivo de apresentar os serviços, permitir contato com clientes e melhorar a presença online da clínica.

O site foi desenvolvido com foco em design moderno, facilidade de navegação e adaptação para celulares.

🚀 Funcionalidades Página inicial com apresentação da clínica Lista de serviços odontológicos Formulário de contato Informações de localização Botão para WhatsApp Site responsivo Área de agendamento 🛠️ Tecnologias utilizadas HTML CSS JavaScript
